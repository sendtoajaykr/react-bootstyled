# React-Bootstyled

Stateless Components built in styled-components

```
npm i react-bootstyled styled-components
```

All in your hand to change color and size

```
<Alert bg="#f00" c="#fff"> Danger </Alert>

<Alert bg="red" c="white" p="5px"> Danger </Alert>
```
